HTML( HyperText Markup Language ) Learning Memo

    2. HTML에서 줄바꿈 태그는 "br"이라는 태그와 "p"라는 태그가 있다.
    br태그는 단순히 키보드에서 enter키 역할만 한다.

    3. 반면 p태그는 지정된 부분을 단락으로 만들어 문장을 구분지어준다.
    p태그의 단점은 단락사이에 간격이 정해져 있다는 것인데 이는 css명령어로 해결 할 수 있다.
    p태그의 대괄호 안에 'p style="margin-top:45px";'라는 명령어를 입력함으로써 간격을 입력 픽셀만큼 정할 수 있다.

    4. HTML에서 각각의 태그를 용도에 맞게 사용한다면 웹검색에서도 유리하고 공공재로서의 역할도 할 수 있다.

    5. img태그에서 어떤 사진을 불러올지 결정하는 것은 속성 src(source)이다.
    src는 어떤 사진을 불러올지 결정하고 width 사진의 크기를 %로 결정한다.

    6. list(목차)의 태그는 li를 사용하고 li태그(list)는 반드시 부모 태그 ul태그(unordered list)를 가지고 있고 ul태그는 반드시 자식 코드인 li코드를 가지고 있다.
    li태그와 ul태그는 말머리 기호를 사용하는 반면 ul태그의 형제인 말고도 ol태그(ordered list)라는 태그도 있는데 ol태그는 목차 앞에 글머리 기호 대신 아래와 같이 문단 번호를 생성다.

  6. 또 다른 부모 자식 태그로는 table(표)태그가 있는데 table 태그, th 태그(표 제목), tr 태그(표 가로줄), td 태그()셀)로 묶여 다닌다.

    • bordercolor는 테두리의 색상을 지정하는 속성. ex(bordercolor="blue") default값은 검정.

    • width(가로 크기)와 height(세로 크기)는 표의 크기를 결정해주는 속성 ex(width=50px width=100%)픽셀 단위로 크기,비율로 줄 수 있다.

    • align은 표 안에 있는 값들을 정렬. ex) (align="center" or "left" or "right")

    • bgcolor속성으로 원하는 태그의 배경색을 지정. ex)(bgcolor="skyblue")default값은 흰색

    • colspan(가로,행 병합), rowspan(세로,열 병합)은 셀을 합병(행/열)시키는 기능
